Cemeteries

    Import photos

    Select photos below. Use “one stone per photo” for the common case; select several shots of the same marker and use “as ONE stone” to group them.

    Stones

    Select a stone to edit it.

    Tag categories

    Categories group tags (Shape, Iconography, …). “Single” categories allow one tag per stone; others allow many. New categories and tags appear immediately in the stone editor and, after publishing, as site filters.
