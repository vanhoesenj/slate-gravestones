Slate Gravestones admin

Cemeteries

    Click the map to set the location, or search below.

    Import photos

    Select photos below. Use “one gravestone per photo” for the common case; select several shots of the same marker and use “as ONE gravestone” to group them.

    Gravestones

    Select a gravestone to edit it.

    Tag categories

    Categories group tags (Shape, Iconography, …). “Single” categories allow one tag per gravestone; others allow many. New categories and tags appear immediately in the gravestone editor and, after publishing, as site filters.

    Gravestone outlines

    Trace silhouettes from new photos, then review below. Only approved outlines are published. The photo is shown beside its traced outline — approve if the silhouette matches the stone's true shape.

    Relief maps — virtual raking light

    Builds an AI depth map per photo, powering the site's raking-light viewer (visitors drag a light across the stone). One-time setup: pip3 install torch transformers (~2GB), then restart the admin app. After building, run Sync images to R2 on the Publish tab.

    Welsh audio

    Reads each Welsh inscription aloud with the Piper open-source Welsh voice and uploads the recording to R2 (needs pip3 install piper-tts once). The site shows a 🔊 button on those inscriptions after the next Export + push.

    Visual constellation

    Computes a visual-similarity embedding per photo (CLIP — same torch transformers install as relief maps). The constellation itself is generated automatically at Export, arranging stones so similar carving sits together.

    Letterforms

    Finds individual carved letters in the enhanced photos (Tesseract OCR: brew install tesseract + pip3 install pytesseract) and crops each one — the site's Letters view becomes a type-specimen sheet of the whole corpus, with per-letter similarity clustering for carver attribution. Review below: ✕ rejects a bad crop, ✎ fixes a misread character.

    Publish

    Drafts land in each gravestone's Inscription field prefixed “[DRAFT]” (translations go to Notes) — only where those fields are empty. Search “[DRAFT]” on the Gravestones tab to find entries awaiting review.
